# @maildev/api

> REST API and WebSocket server for MailDev.

Part of the [MailDev](https://github.com/maildev/maildev) monorepo. This package exposes the HTTP REST API and WebSocket stream for reading emails, and optionally serves the web UI and the integrated MCP endpoint.

Most users should install the [`maildev`](https://www.npmjs.com/package/maildev) CLI rather than depending on this package directly.

## Install

```bash
npm install @maildev/api
```

## Documentation

See the [main MailDev README](https://github.com/maildev/maildev#readme) for full documentation.
