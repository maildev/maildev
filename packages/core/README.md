# @maildev/core

> Core types, utilities, and storage abstraction for MailDev.

Part of the [MailDev](https://github.com/maildev/maildev) monorepo. This package provides the shared email model, storage interfaces, and in-memory storage used across the other `@maildev/*` packages.

Most users should install the [`maildev`](https://www.npmjs.com/package/maildev) CLI rather than depending on this package directly.

## Install

```bash
npm install @maildev/core
```

## Documentation

See the [main MailDev README](https://github.com/maildev/maildev#readme) for full documentation.
