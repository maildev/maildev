# @maildev/smtp

> SMTP server for MailDev.

Part of the [MailDev](https://github.com/maildev/maildev) monorepo. This package implements the SMTP server that receives mail and persists it through a `@maildev/core` storage backend.

Most users should install the [`maildev`](https://www.npmjs.com/package/maildev) CLI rather than depending on this package directly.

## Install

```bash
npm install @maildev/smtp
```

## Documentation

See the [main MailDev README](https://github.com/maildev/maildev#readme) for full documentation.
