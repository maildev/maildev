# @maildev/ui

> React-based web UI for MailDev.

Part of the [MailDev](https://github.com/maildev/maildev) monorepo. This package contains the React web interface for reading and testing emails, along with a small server helper (`@maildev/ui/server`) for serving the built static assets.

Most users should install the [`maildev`](https://www.npmjs.com/package/maildev) CLI rather than depending on this package directly.

## Install

```bash
npm install @maildev/ui
```

## Documentation

See the [main MailDev README](https://github.com/maildev/maildev#readme) for full documentation.
