# @maildev/mcp

> MCP (Model Context Protocol) server for Claude integration with MailDev.

Part of the [MailDev](https://github.com/maildev/maildev) monorepo. This package provides the MCP server that lets Claude and other MCP clients search, read, and manage development emails — available both as a standalone stdio server (`maildev-mcp`) and embedded in the `@maildev/api` HTTP server at `/mcp`.

Most users should install the [`maildev`](https://www.npmjs.com/package/maildev) CLI rather than depending on this package directly.

## Install

```bash
npm install @maildev/mcp
```

## Documentation

See the [main MailDev README](https://github.com/maildev/maildev#readme) for MCP setup and usage documentation.
